1.Installation-notes

- Installed Oracle VirtualBox on Windows PC.
- Created a new VM named "VP COMPUTER".
- Assigned 8GB RAM and 60GB virtual hard disk.
- Attached Windows 10 ISO in VM storage settings.
- Followed the Windows 10 setup wizard.
- Completed initial setup: region, keyboard, username, and password.
- Installed VirtualBox Guest Additions for better performance.
- Optional: Took a snapshot after installation for backup.

- Installation was smooth, no errors faced.
- VM performance is good with 8GB RAM and 60GB disk.
- Screenshots were taken during installation for documentation.
